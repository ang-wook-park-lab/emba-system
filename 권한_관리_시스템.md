## 🎯 관리자가 할 수 있는 작업

### 1. 계정 관리
1. 좌측 사이드바에서 **"계정 관리"** 클릭
2. 사용자 목록 확인
3. 수정하려는 사용자의 **"수정"** 버튼 클릭
4. 다음 항목 수정 가능:
   - 부서
   - 직급/직책
   - 권한 (user/approver/admin)
   - 활성화 상태
5. **"저장"** 버튼으로 변경사항 저장

### 2. 권한 변경
- **user → approver**: 승인 권한 부여
- **approver → admin**: 전체 관리 권한 부여
- **admin → user**: 일반 사용자로 강등

### 3. 프로젝트 관리
1. **"프로젝트 관리"** 페이지 접속
2. **"프로젝트 추가"** 버튼으로 새 프로젝트 생성
3. 각 프로젝트의 **"수정"** 버튼으로 정보 변경
4. 각 프로젝트의 **"삭제"** 버튼으로 프로젝트 삭제

### 4. 승인 처리
1. **"내정보"** 페이지의 **"승인 대기 목록"** 탭
2. 승인 요청 확인
3. **"승인"** 또는 **"거절"** 버튼으로 처리

## 🔒 보안 기능

### 접근 제어
- 각 페이지는 사용자 권한에 따라 접근 제한
- 권한이 없는 페이지 접근 시 "접근 권한이 없습니다" 메시지 표시
- 관리자 전용 버튼은 관리자에게만 표시

### 자기 보호
- 관리자는 자기 자신의 계정을 삭제할 수 없음
- 로그인된 사용자는 자동으로 권한 확인
